🔰 Overview

  • CSV পড়বে এবং প্রতিটি রো অনুযায়ী ইমেজ ফাইলের নাম বদলাবে (Filename → Title ভিত্তিক)।
  • Windows-এর illegal character গুলো (<>:"/\|?*) নাম থেকে সরিয়ে দেবে এবং ".." কে "_" করবে।
  • ফোল্ডারের আসল ফাইলের extension আগের মতই থাকবে (.jpg/.png), তবে CSV এর “New Filename” কলামে .eps লেখা থাকবে।
  • Progress bar দেখাবে এবং শেষে summary দেবে।

🧩 Requirements

  • Windows 10/11
  • Python 3.9+ (Tkinter সাধারণত ডিফল্টে থাকে)
  • pandas প্যাকেজ
    pip install pandas
  • CSV ফাইলে কমপক্ষে এই কলামগুলো থাকতে হবে:
    • Filename (উদাহরণ: image_01.jpg)
    • Title (উদাহরণ: Modern Flat Icon Set)

📁 CSV Format (Example)

Filename,Title image_01.jpg,Modern Flat Icon Set image_02.png,Minimal Line Icons

➡ রান করার পরে স্ক্রিপ্ট CSV-তে একটি নতুন কলাম যোগ করবে: New Filename (Title থেকে জেনারেট করা নাম + .eps)

⚙️ How to Run

  1. স্ক্রিপ্ট ফাইল (smart_name_changer.py) এবং আপনার CSV/ইমেজ ফোল্ডার প্রস্তুত রাখুন।
  2. স্ক্রিপ্ট চালান:
    • ডাবল ক্লিক (যদি .py ফাইল Python এর সাথে অ্যাসোসিয়েটেড থাকে), অথবা
    • Command Prompt / PowerShell থেকে:
      python smart_name_changer.py
  3. অ্যাপ খুললে:
    • “Select Image Folder” → ইমেজ ফোল্ডার সিলেক্ট করুন।
    • “Select CSV File” → CSV ফাইল সিলেক্ট করুন।
    • “Change Names” বাটনে ক্লিক করুন।
  4. অ্যাপ CSV লোড করবে, ফাইল rename করবে এবং CSV আপডেট করবে। Progress bar দেখাবে এবং শেষে success message দিবে।

🧠 Naming Rules

  • Illegal characters সরানো হবে: <>:"/\|?*
  • ".." → "_" এ রূপান্তর হবে।
  • ফোল্ডারের ফাইলের আসল extension বজায় থাকবে।
  • CSV-র “New Filename” কলাম সবসময় .eps দিয়ে শেষ হবে।
// উদাহরণ Title: Modern: Flat/Icon..Set? Sanitized: Modern FlatIcon_Set Physical file: Modern FlatIcon_Set.jpg CSV [New Filename]: Modern FlatIcon_Set.eps

✅ Best Practices

  • Title গুলো ইউনিক রাখুন।
  • CSV এর Filename কলামে ফোল্ডারের ফাইলের নামের সাথে ১০০% মিল থাকতে হবে।
  • CSV UTF-8 এ সেভ করুন।
  • রান করার আগে ইমেজ ফোল্ডারের ব্যাকআপ রাখুন।
  • বড় ব্যাচে রান করার আগে ২–৩ ফাইলে টেস্ট করুন।

🛠️ Troubleshooting

  • “Please select a valid image folder.” → ফোল্ডার চেক করুন।
  • “Please select a valid CSV file.” → CSV path সঠিক কিনা চেক করুন।
  • “The file 'xxx' could not be found...” → Filename এবং ফোল্ডারের নাম মিল কিনা চেক করুন।
  • PermissionError → ফাইল অন্য প্রোগ্রামে খোলা নেই কিনা চেক করুন।
  • Unicode/Encoding Error → CSV UTF-8 format এ সেভ করুন।
  • Duplicate Title → Title ইউনিক করুন বা শেষে -1/-2 যোগ করুন।

🧪 Verification

  • ফোল্ডারের ফাইলগুলোর নাম Title ভিত্তিক নতুন নামে দেখা যাবে।
  • CSV-তে “New Filename” কলাম তৈরি হবে এবং সেখানে .eps নাম থাকবে।
  • অ্যাপে success message দেখাবে।

❓ FAQ

  • CSV-র “New Filename” কেন .eps? → Vector workflow জন্য। Physical file extension আগের মতো থাকে।
  • একই Title থাকলে কী হবে? → Title ইউনিক করুন, প্রয়োজনে "-01", "-02" যোগ করুন।
  • Notepad-এ লিংক ক্লিক হয় না? → Notepad++/VS Code/Browser/Word ব্যবহার করুন। Local file path হলে file:///C:/path ফরম্যাট ব্যবহার করুন।
