📘 Vectorizer Pro — User Manual

Version: 1.0
Developed by: Mostafizar Rahman • Contact: Telegram: @mostafizarfiz

🔰 Overview

Vectorizer Pro একটি অটোমেশন টুল যা আপনার কম্পিউটারের input ফোল্ডার থেকে JPG, JPEG, PNG ফাইলগুলো নিয়ে স্বয়ংক্রিয়ভাবে Vectorizer.AI ওয়েবসাইটে আপলোড করে EPS ফাইল আকারে output ফোল্ডারে ডাউনলোড করে।

  • Input ফোল্ডারে থাকা সমস্ত ইমেজ প্রসেস করা হবে।
  • Browser session একবার সংরক্ষণ করলে পুনরায় লগইন দরকার হবে না।
  • Progress bar এবং live status label দিয়ে কাজের অবস্থা দেখাবে।

🧩 Requirements

  • Windows 10/11 বা নতুন
  • Python 3.8+
  • Playwright লাইব্রেরি: python -m pip install playwright এবং python -m playwright install
  • Vectorizer.AI Premium অ্যাকাউন্ট
  • Internet connection

📁 Folder Structure

প্রজেক্ট ফোল্ডার উদাহরণ:


E:\Vectorizer-Pro\
    ├── input\        (JPG, JPEG, PNG ফাইল)
    ├── output\       (EPS ফাইল সংরক্ষণের জন্য)
    ├── VectorizerPro.py
    └── session.json  (First Time Run করলে Automatic Create হবে)

⚙️ Setup Steps

  1. Python ইনস্টল করুন এবং "Add Python to PATH" সিলেক্ট করুন।
  2. প্রজেক্ট ফোল্ডার তৈরি করুন এবং input/output ফোল্ডার তৈরি করুন।
  3. Playwright ইনস্টল করুন এবং browser binaries ইনস্টল করুন:
    pip install playwright python -m playwright install
  4. VectorizerPro.py ফাইল ফোল্ডারে রাখুন।

🚀 How to Use

  1. PowerShell/Terminal খুলুন এবং প্রজেক্ট ফোল্ডারে যান।
  2. স্ক্রিপ্ট চালান:
    python VectorizerPro.py
  3. প্রথমবার session সংরক্ষণ:
    • "1. Save Session" বাটনে ক্লিক করুন।
    • Browser খুলবে। আপনার Vectorizer.AI অ্যাকাউন্টে লগইন করুন।
    • লগইন শেষে browser বন্ধ করুন। session.json ফাইল তৈরি হবে।
  4. Input ফোল্ডারে ইমেজ রাখুন।
  5. "2. Process Images" বাটনে ক্লিক করুন। স্ক্রিপ্ট সব ইমেজ EPS এ প্রসেস করে output ফোল্ডারে সংরক্ষণ করবে।

📝 Notes & Tips

  • Input ফোল্ডার খালি থাকলে বা session.json না থাকলে কাজ করবে না।
  • Session সংরক্ষণ একবার করতে হবে।
  • EPS ফাইল ডাউনলোডের সময় browser visible থাকবে।
  • বড় batch এর আগে কয়েকটি ফাইল দিয়ে টেস্ট করুন।
  • সবসময় input ফোল্ডারের ব্যাকআপ রাখুন।

🛠️ Troubleshooting

  • Browser open হচ্ছে না → Playwright সঠিকভাবে ইনস্টল হয়েছে কিনা চেক করুন।
  • Session file not found → "Save Session" বাটন ক্লিক করুন।
  • Input/Output ফোল্ডার নেই → সঠিক ফোল্ডার সিলেক্ট করুন।
  • Download fail → Internet এবং Vectorizer.AI Premium account চেক করুন।
  • PermissionError → ফাইল অন্য প্রোগ্রামে খোলা নেই কিনা চেক করুন।

❓ FAQ

  • একই নামের ছবি আছে, কী হবে? → Output ফোল্ডারে duplicate হলে browser auto rename করতে পারে।
  • EPS ফাইল কোথায় সংরক্ষিত হবে? → Output ফোল্ডারে।
  • কত ছবি একবারে প্রসেস করা যাবে? → কম্পিউটার resources অনুযায়ী।
