🎨 UniteByColor.jsx Manual

📌 Overview

Same color objects automatically merge করে (Pathfinder Unite)।

📁 Install

Scripts → Illustrator/Presets/Scripts
Actions → Illustrator/Presets/Actions

👉 Script + Action copy করে paste করুন, তারপর Illustrator restart করুন

▶️ Run

File → Scripts → UniteByColor

🧩 Action

Window → Actions → Load → Expand_Divide_Clean_Unity

🔄 Workflow

  1. Script run
  2. Action run

🎯 Features

  • Auto color detect
  • Auto merge
  • RGB / CMYK / Spot support

⚠️ Notes

  • Only filled objects
  • Gradient সমস্যা করতে পারে

📌 Summary

✔ Fast
✔ Easy
✔ Clean output
