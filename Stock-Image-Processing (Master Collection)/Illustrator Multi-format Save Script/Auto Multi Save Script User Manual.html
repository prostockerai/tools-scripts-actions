🔰 Overview

এই স্ক্রিপ্টটি Illustrator এ ব্যবহার করা হয় একটি নতুন ডকুমেন্ট তৈরি, ডিজাইন স্কেল এবং Multi-format (JPG, PNG, AI, EPS 10) এ সেভ করার জন্য।

  • নতুন ডকুমেন্টে ডিজাইনটি কেন্দ্রে বসানো হবে।
  • JPG এর জন্য সাদা ব্যাকগ্রাউন্ড তৈরি হবে, PNG ট্রান্সপারেন্ট থাকবে।
  • AI ও EPS 10 ফাইলও একই ডকুমেন্ট থেকে তৈরি হবে।
  • Action ফাইলগুলো ব্যবহার করলে Object Expand & Divide করে ফাইনাল Multi-format ফাইলগুলো সেভ করা সহজ হবে।

🧩 Requirements

  • Windows 10/11 বা নতুন
  • Adobe Illustrator 2025
  • Illustrator Script: Auto Multi-Save Script.jsx
  • Action ফাইল: Expant and Divide.aia & Auto Multi Save Script.aia
  • Internet connection (Optional, যদি Online resources দরকার হয়)

📁 Folder & File Setup

Script এবং Action ফাইলগুলোর সঠিক লোকেশন:

  • Script ফোল্ডারে স্ক্রিপ্ট রাখুন:
    C:\Program Files\Adobe\Adobe Illustrator 2025\Presets\en_GB\Scripts\Auto Multi-Save Script.jsx
  • Action ফোল্ডারে Action ফাইল রাখুন:
    C:\Program Files\Adobe\Adobe Illustrator 2025\Presets\en_GB\Actions\Expant and Divide.aia
    C:\Program Files\Adobe\Adobe Illustrator 2025\Presets\en_GB\Actions\Auto Multi Save Script.aia

ফোল্ডার স্ট্রাকচার ঠিক রাখলে স্ক্রিপ্ট ও Action seamless কাজ করবে।

⚙️ How to Use

🟢 Option 01 — Direct Script Run

  1. Illustrator ওপেন করুন এবং আপনার ডিজাইন ফাইলটি Open করুন।
  2. ডিজাইনটি সিলেক্ট করুন।
  3. Run Script: File > Scripts > Other Script… নির্বাচন করুন এবং Auto Multi-Save Script.jsx চালান।
  4. স্ক্রিপ্ট পপ-আপ উইন্ডো আসবে। এখানে করুন:
    • Select Format(s): JPG, PNG, AI, EPS 10 নির্বাচন করুন।
    • Document এবং Design Dimensions (px): প্রয়োজনীয় উচ্চতা ও প্রস্থ লিখুন।
    • File Name: ফাইলের নাম লিখুন (ডিফল্ট: মূল ফাইলের নাম)।
    • Save Folder: যেখানে ফাইল সেভ হবে তা নির্বাচন করুন।
  5. Save বাটনে ক্লিক করুন। স্ক্রিপ্ট নির্বাচিত ফরম্যাটে ফাইলগুলো সেভ করবে। ✅

🟡 Option 02 — Using Action Panel + Script

  1. Illustrator ওপেন করুন এবং আপনার ডিজাইন ফাইলটি লোড করুন।
  2. Action Panel খুলুন (Window > Actions) এবং নিম্নোক্ত Action ফাইলগুলো লোড/Setup করুন:
    • Expant and Divide.aia — ডিজাইনের Object গুলো Expand এবং Divide করবে।
    • Auto Multi Save Script.aia — Multi-format ফাইল সেভের জন্য স্ক্রিপ্ট চালাবে।
  3. যদি ডিজাইনের Objectগুলোকে Expand বা Divide করা প্রয়োজন হয়, তবে Expant and Divide Action চালান।
  4. সবকিছু ঠিক থাকলে Auto Multi Save Script Action চালান।
  5. Save Folder Selection: যদি কোনো ফোল্ডার নির্বাচিত না করা হয়, স্ক্রিপ্টটি স্বয়ংক্রিয়ভাবে Desktop-এ "AI_Exports" নামে একটি ফোল্ডার তৈরি করবে এবং সব ফাইল সেখানে সেভ হবে। ✅
  6. Action চালানোর পর, সব ফাইল নির্বাচিত ফরম্যাটে সেভ হয়ে যাবে।

🧩 Action Panel Setup

Action ফাইলগুলো ব্যবহার করলে অনেক কাজ স্বয়ংক্রিয়ভাবে হবে।

  • Expant and Divide.aia — ডিজাইনের ভিতরের Object গুলো Expand & Divide করে।
  • Auto Multi Save Script.aia — স্ক্রিপ্ট রান করে Multi-format ফাইল সেভ করবে।
  • Action Panel (Window > Actions) খুলে প্রতিটি Action ইনস্টল করুন।

📝 Notes & Tips

  • ডিফল্ট Save Folder পরবর্তীতে স্মরণ থাকবে।
  • Design স্কেলিং পিক্সেল এক্স্যাক্ট হতে হবে যদি প্রিসাইস মাপ চান।
  • Batch ফাইল প্রোসেস করার আগে কয়েকটি টেস্ট করুন।
  • সবসময় মূল ডিজাইনের ব্যাকআপ রাখুন।

🛠️ Troubleshooting

  • EPS বা AI সেভ হচ্ছে না → Compatibility এবং Illustrator update চেক করুন।
  • Action চালাচ্ছে না → Action ফাইল সঠিক ফোল্ডারে আছে কিনা চেক করুন।
  • Script Error → Illustrator restart করুন এবং আবার চেষ্টা করুন।

❓ FAQ

  • একই নামের ফাইল? → Output ফোল্ডারে duplicate হলে auto rename হবে।
  • EPS কোথায় সেভ হবে? → Save Folder-এ।
  • কতগুলো একসাথে প্রসেস হবে? → System resource অনুযায়ী।
