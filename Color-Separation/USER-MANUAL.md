# 🎨 ColorSep Pro.jsx — User Manual

---

## 📌 Overview
**ColorSep Pro.jsx** একটি Adobe Illustrator script যা design কে automatically **color separation** করে print-ready output তৈরি করে।

👉 এছাড়া `Color Seperation.aia` Action ব্যবহার করলে workflow আরও fast ও automated হয়।
```
👉 Script: ColorSep Pro.jsx
👉 Action: Color Seperation.aia
```

---

## 📁 Installation (Windows)

### 📂 Script Location:
```
C:\Program Files\Adobe\Adobe Illustrator 2025\Presets\en_GB\Scripts
```

### 📂 Action Location:
```
C:\Program Files\Adobe\Adobe Illustrator 2025\Presets\en_GB\Actions
```

### ✅ Steps:
1. `ColorSep Pro.jsx` file copy করুন  
2. Scripts folder-এ paste করুন  
3. `Color Seperation.aia` file copy করুন  
4. Actions folder-এ paste করুন  
5. Adobe Illustrator restart করুন  

---

## 🍎 Installation (Mac)

### 📂 Script Location:
```
/Applications/Adobe Illustrator 2025/Presets/en_GB/Scripts
```

### 📂 Action Location:
```
/Applications/Adobe Illustrator 2025/Presets/en_GB/Actions
```

👉 একইভাবে `.jsx` এবং `.aia` file paste করে Illustrator restart করুন  

---

## ⚙️ How to Run Script

1. Adobe Illustrator open করুন  
2. আপনার design file open করুন  
3. Menu →  

File → Scripts → ColorSep Pro

4. Script run হবে এবং color separation শুরু হবে  

---

## 🧩 How to Setup Action (Color Seperation.aia)

### ✅ Steps:
1. Illustrator open করুন  
2. **Window → Actions** panel open করুন  
3. Actions panel menu (☰) click করুন  
4. **Load Actions** select করুন  
5. `Color Seperation.aia` file select করুন  
6. Action load হয়ে যাবে  

---

## ▶️ How to Use Action

1. Design open করুন  
2. Object select করুন  
3. Actions panel থেকে  
**Color Seperation** action select করুন  
4. ▶️ (Play) button চাপুন  
5. Automatic process run হবে  

---

## 🎯 What This Script & Action Do

- 🎨 Design থেকে colors detect করে  
- 🧩 প্রতিটি color আলাদা layer/group-এ ভাগ করে  
- 🖨️ Print-ready separation তৈরি করে  
- ⚡ Manual কাজ কমিয়ে automation দেয়  

---

## 🔄 Workflow

1. Design open  
2. Object select  
3. Script অথবা Action run  
4. Auto color separation  
5. Output check  

---

## 🛠️ Requirements

- Adobe Illustrator 2020 বা newer  
- Vector design (recommended)  

---

## ⚠️ Notes

- Raster image হলে proper কাজ নাও করতে পারে  
- Complex gradient থাকলে expand/flatten করা ভালো  
- কাজের আগে backup রাখা recommended
