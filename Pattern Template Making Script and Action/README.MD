
# 📦 Illustrator Pattern Template Scripts

Collection of custom **Adobe Illustrator scripts & actions** for automating pattern template workflows.

⚡ **One-click workflow:**

👉 Scripts background-এ

👉 Actions panel থেকে শুধু ▶ Play

---

# 📥 Installation & Setup

## 🔹 Install Scripts (.jsx)

### 🪟 Windows

```
C:\Program Files\Adobe\Adobe Illustrator [version]\Presets\en_GB\Scripts
```

### 🍎 MacOS

```
/Applications/Adobe Illustrator [version]/Presets.localized/en_GB/Scripts
```

👉 Restart Illustrator

---

## 🔹 Install Actions (.aia)

1. Open Illustrator
2. Window → Actions
3. Menu (☰) → Load Actions
4. Select `.aia`

---

⚙️ How to Use (Important)

👉 এই system-এ সব script action-এর সাথে connected

✔ Basic Workflow:
Select your main design / artwork
Open Actions Panel
Select the required action
Click ▶ Play button

✅ No need to run scripts manually
✅ Everything runs automatically via actions
