## 🎯 Overview

**Pantone Color Book Builder** is an Adobe Illustrator script designed to automatically generate a **print-ready Pantone color swatch book** for accurate spot color matching in print and RIP workflows.

This tool helps designers and print professionals quickly create a **physical reference book** tailored to their printer, media, and workflow.

---

## ✨ Features

* ⚡ One-click Pantone book generation
* 🎨 Organized color layout (Pantone order)
* 🖨️ Print-ready output (RIP compatible)
* 📐 Cut guide support (laser cutting optional)
* 🧩 Fully customizable template
* 💼 Professional production workflow ready

## ⚙️ Installation

### 🔹 Step 1: Install Script

Copy the script file to your Illustrator Scripts folder:

**Windows:**

```
C:\Program Files\Adobe\Adobe Illustrator [version]\Presets\en_US\Scripts
```

**MacOS:**

```
/Applications/Adobe Illustrator [version]/Presets.localized/en_US/Scripts
```

---

### 🔹 Step 2: Restart Illustrator

After copying the script, restart Adobe Illustrator.

## 🚀 How to Use

1. Open Illustrator

2. Open the provided template file:

   * `Pantone-YESCO Template CC2014.ai`
   * or `Pantone-YESCO Template CS6.ai`

3. Customize:

   * Replace logo/title
   * Add printer name
   * Add media name

4. Run the script:

```
File → Scripts → Pantone Book Maker.jsx
```

5. Wait patiently ⏳

   * Script is CPU intensive
   * Illustrator may appear frozen

6. Review output:

   * Remove unused labels
   * Adjust layout if needed

7. Print & bind your Pantone book 🎉

---

## ⚠️ Important Notes

* ❗ Do NOT change layer/group structure before running
* ⚙️ Script is single-threaded (takes time)
* ✂️ Guide lines are cut lines (for laser cutting)
* 🎨 Colors follow Pantone order (not exact page index)
