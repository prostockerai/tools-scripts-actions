# 🎨 Illustrator — Pantone Color Books Manual

---

## 📌 Overview
নতুন Adobe Illustrator (2025–2026)-এ Pantone Color Books default থাকে না (license issue এর কারণে)।  
তাই `.acb` files manually add করতে হয়।

---

## 📁 Install (Windows)

```
C:\Program Files\Adobe\Adobe Illustrator 2025\Presets\en_GB\Swatches\Color Books
```

## 🍎 Install (Mac)

```
/Applications/Adobe Illustrator 2025/Presets/en_GB/Swatches/Color Books
```

---


### ✅ Steps:
1. Color Books (`.acb` files) ডাউনলোড করুন  
2. উপরের folder location-এ paste করুন  
3. Adobe Illustrator restart করুন  

---

## 🎯 Pantone কী?
Pantone একটি standard color system যা exact color, print consistency এবং branding নিশ্চিত করে।

---

## ⚙️ How to Use

1. Window → Swatches  
2. Menu (☰) → Color Books  
3. Pantone library select করুন  
4. Object-এ apply করুন
